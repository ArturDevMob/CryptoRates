# Crypto Rates

Приложение позволяет следить за текущим курсом криптовалют

Были использованы следующие технологии и инструменты:

* Kotlin
* MVVM
* LiveData
* Dagger 2
* RxJava 2
* Room
* Retrofit 2
* Kotlin synthetic
* Picasso
